## MKVToolNix-Automatisierung 1.0.0

Erster stabiler Release der lokalen Windows-App zur Automatisierung von MKVToolNix-Muxing fuer Serienepisoden aus MediathekView-Downloads.

### Enthalten

- Einzelmodus fuer eine Folge mit automatischer Erkennung und manueller Korrekturmoeglichkeit
- Batchmodus zum Scannen und Verarbeiten ganzer Serien-Downloadordner
- automatische Erkennung von Hauptvideo, Audiodeskription, Untertiteln und TXT-Metadatenanhaengen
- Archivabgleich gegen die vorhandene Serienbibliothek
- Wiederverwendung vorhandener Bibliotheksdateien, wenn Inhalte ergaenzt statt blind ersetzt werden koennen
- TVDB-Abgleich inklusive manueller Pruefung
- Batch-Protokoll, Fortschritt, Abbruchpfade und Cleanup nach erfolgreicher Verarbeitung
- portable lokale Einstellungen unter `Data\settings.json`

### Voraussetzungen

- Windows
- .NET 9 Desktop Runtime
- MKVToolNix / `mkvmerge.exe`
- optional: `ffprobe.exe` fuer genauere Laufzeitinformationen

### Download

Die bereitgestellte `.exe` ist framework-dependent und enthaelt daher nicht die komplette .NET-Laufzeit. Das haelt die Datei deutlich kleiner; dafuer muss die passende .NET Desktop Runtime installiert sein.
