## MKVToolNix-Automatisierung 1.0.0

Erster stabiler Release der lokalen Windows-App zur Automatisierung von MKVToolNix-Muxing für Serienepisoden aus MediathekView-Downloads.

### Enthalten

- Einzelmodus für eine Folge mit automatischer Erkennung und manueller Korrekturmöglichkeit
- Batchmodus zum Scannen und Verarbeiten ganzer Serien-Downloadordner
- automatische Erkennung von Hauptvideo, Audiodeskription, Untertiteln und TXT-Metadatenanhängen
- Archivabgleich gegen die vorhandene Serienbibliothek
- Wiederverwendung vorhandener Bibliotheksdateien, wenn Inhalte ergänzt statt blind ersetzt werden können
- TVDB-Abgleich inklusive manueller Prüfung
- Batch-Protokoll, Fortschritt, Abbruchpfade und Cleanup nach erfolgreicher Verarbeitung
- portable lokale Einstellungen unter `Data\settings.json`

### Voraussetzungen

- Windows
- .NET 9 Desktop Runtime
- MKVToolNix / `mkvmerge.exe`
- optional: `ffprobe.exe` für genauere Laufzeitinformationen

### Download

Die bereitgestellte `.exe` ist framework-dependent und enthält daher nicht die komplette .NET-Laufzeit. Das hält die Datei deutlich kleiner; dafür muss die passende .NET Desktop Runtime installiert sein.
